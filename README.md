# 🎮 Memory Card Game

A simple and interactive Memory Card Matching Game built using HTML, CSS, and JavaScript. The objective of the game is to test and improve memory skills by matching pairs of cards within the least number of moves.

## 🧠 How It Works
- Cards are placed face down in a grid
- Click on two cards to flip them
- If they match, they stay flipped
- If not, they flip back after a short delay
- The game continues until all pairs are matched

## ✨ Features
- Smooth card flip animations
- Move counter / score tracking
- Responsive design (works on desktop & mobile)
- Restart / reset game option
- Simple and user-friendly interface

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 🎯 Purpose
This project was created to practice core JavaScript concepts like:
- DOM manipulation
- Event handling
- Game logic implementation
